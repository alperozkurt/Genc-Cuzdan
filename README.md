[TR]
Bu proje, ortaokul, lise ve üniversite düzeyindeki gençlerin finansal okuryazarlık becerilerini geliştirmeyi amaçlayan bir mobil FinTech uygulamasıdır.
Uygulama, kullanıcıların gelir-gider takibini kolaylaştırır, kişiye özel tasarruf önerileri üretir ve hedef odaklı birikim planları oluşturur.

[EN]
This project is a mobile FinTech application that aims to improve the financial literacy skills of middle school, high school, and university students. The application facilitates user income and expense tracking, generates personalized savings recommendations, and creates goal-oriented savings plans.
